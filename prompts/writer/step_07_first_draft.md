## 원본 보존 원칙

시나리오 1고는 트리트먼트(Step 5)를 기반으로 쓰되, 원본 시놉시스의 요소가 빠지지 않았는지 확인해야 한다. Step 0 사전 분석의 평가를 기준으로 삼는다.

### 1. Step 0 강점(섹션 2)에 해당하는 장면과 디테일은 반드시 시나리오에 포함한다
- Step 0에서 "반드시 유지해야 할 요소"로 지목된 것들이 트리트먼트를 거쳐 시나리오까지 살아 있어야 한다.
- 트리트먼트에 있는 강점 요소를 시나리오로 옮기면서 빠뜨리지 않는다.

### 2. Step 0 보완점(섹션 3)에 따라 개선된 부분은 그 개선을 유지한다
- 트리트먼트에서 보완된 내용(심리적 여정 심화, 복선 배치, 인물 입체화 등)이 시나리오에서도 살아 있어야 한다.
- 시나리오 형식으로 변환하면서 보완된 내용을 다시 약화시키지 않는다.

### 3. 트리트먼트에 있는 장면과 디테일을 시나리오에서 삭제하지 않는다
- 트리트먼트는 시나리오의 설계도다. 설계도에 있는 장면을 집필 과정에서 임의로 빼지 않는다.
- 시나리오 형식으로 전환하면서 **확장**하는 것은 좋지만, **축소**하거나 **삭제**하지 않는다.
- 트리트먼트의 산문 묘사를 대지문/소지문/대사로 풀어내는 것이 이 Step의 핵심 작업이다.

## 한국 시나리오 형식

한국 시나리오는 할리우드 시나리오와 형식이 다르다. 할리우드 형식이 아니라 **한국 형식으로** 써야 한다.

### 할리우드와 한국 형식의 차이

| 항목 | 할리우드 | 한국 |
|------|---------|------|
| 씬 넘버 | 촬영 대본에서만 붙인다 | 초고부터 S#으로 붙인다 |
| 씬 헤딩 | `INT. LOCATION - DAY` (전부 대문자) | `S#1. 장소 / 밤` (볼드체, 13~14pt) |
| 지문 구분 | 구분 없이 모두 "action"이다 | **대지문**과 **소지문**을 구분한다 |
| 대사 정렬 | 페이지 중앙에 좁은 칼럼으로 정렬한다 | **왼쪽 정렬 + 들여쓰기**로 쓴다 |
| 인물명 | 대문자로 중앙 정렬한다 | 볼드체로 왼쪽 들여쓰기한다 |
| 폰트 | Courier 12pt (고정폭) | 굴림 또는 돋움 10~11pt (가변폭) |
| 1페이지 = 시간 | 약 1분 | **약 1분 30초** |
| 주요 지표 | 페이지 수 | **씬 수** (페이지 수는 보조 지표) |

### 씬 헤딩 (S#)

씬 헤딩은 다음과 같이 쓴다. 씬 넘버를 앞에 붙이고, 장소와 시간을 적는다.

```
S#1. 민준의 원룸 / 밤
```

- 장소가 세분화될 때는 하이픈으로 구분한다: `S#12. 태준의 집 - 거실 / 낮`
- 시간은 한국어로 쓴다: 낮, 밤, 새벽, 저녁, 아침
- 같은 장소에서 시간이 이어질 때는 `S#13. 같은 장소 / 잠시 후`로 쓸 수 있다.

### 대지문과 소지문

한국 시나리오는 지문을 두 종류로 구분한다. 이 구분을 지켜서 써야 한다.

**대지문(大指文)**은 씬 헤딩 바로 아래에 쓴다. 그 장소의 전체적인 공간, 분위기, 상황을 묘사한다. 카메라가 장소에 처음 들어왔을 때 보이는 풍경이다.

**소지문(小指文)**은 대사 사이사이에 쓴다. 특정 인물의 행동, 표정, 반응을 묘사한다. 대사를 주고받는 도중에 인물이 무엇을 하는지 보여준다.

```
S#28. 찜질방 / 밤

  바닥에 누운 사람들. 코 고는 소리가 여기저기서 들린다.
  누런 형광등이 낮은 천장에 붙어 있다.
  민준이 한쪽 구석에 누워 식혜를 홀짝인다.

      태준   야, 민준아! 여기서 뭐 해?

  민준이 몸을 일으킨다. 태준이 성큼 다가와 민준의 어깨를 잡는다.

      민준   (당황하며) 그냥... 좀 쉬러 왔어.

  태준의 눈이 민준의 초췌한 얼굴을 훑는다.
```

위 예시에서:
- "바닥에 누운 사람들 ~ 식혜를 홀짝인다"는 **대지문**이다. 장소의 전체 풍경을 보여준다.
- "민준이 몸을 일으킨다", "태준의 눈이 ~ 훑는다"는 **소지문**이다. 특정 인물의 행동을 보여준다.

### 대사 형식

대사는 **왼쪽 들여쓰기**로 쓴다. 할리우드처럼 중앙 정렬하지 않는다.

```
      민준   지우야, 나 너한테 할 말이 있어.

      지우   (웃으며) 뭔데?

      민준   (말을 멈추고, 핸드폰을 내려놓는다)
             ...우리 만나자.
```

- 인물명은 왼쪽에서 일정 간격을 두고 쓴다.
- 대사는 인물명과 같은 줄에 쓰거나, 다음 줄에 들여쓰기하여 쓴다.
- 지시(톤, 행동)는 괄호 안에 넣는다: `(읽으며)`, `(작게)`, `(돌아서며)`
- 긴 대사가 이어질 때는 다음 줄로 넘기되, 들여쓰기를 유지한다.

### 여백의 미학 (餘白의 美學)

한국 시나리오는 **모든 것을 쓰지 않는다.** 감독과 배우가 해석할 공간을 남겨두어야 한다.

실천 방법:
- 지문은 3~5줄마다 줄을 바꿔서 시각적 여백을 만든다. 빽빽한 텍스트 블록은 피한다.
- 감정을 설명하지 않는다. 행동만 쓰면 감정은 배우와 감독이 채운다.
- 소품이나 세트의 세부사항을 과도하게 지정하지 않는다. "따뜻한 느낌의 거실"이면 충분하다.
- 인물의 반응을 일일이 쓰지 않는다. 대사 뒤 빈 줄 하나가 침묵을 표현한다.

## 분할 집필 절차

### 호출 1: 1막 집필

**입력**: 트리트먼트(1막) + 씬 리스트(1막 씬들) + 이미지 시스템 + 캐릭터
**출력**: `step_07_first_draft_act1.md`

집필을 마친 뒤, 파일 마지막에 다음 요약을 추가한다:

```markdown
---
## [이전 막 요약 — 2막 전달용]
- **마지막 씬 상황**: ...
- **캐릭터 감정 상태**: ...
- **미해결 복선**: ...
- **이미지 모티프 상태**: ...
- **마지막 대사/이미지**: ...
```

**→ 사용자 점검.** 1막 집필 결과를 사용자에게 제시하고, 다음 중 하나를 선택하도록 한다:
- **승인**: 2막 전반 집필로 넘어간다.
- **수정 요청**: 사용자의 피드백을 반영하여 1막을 수정한 뒤 다시 점검한다.

### 호출 2: 2막 전반(2a) 집필

**입력**: 1막 요약 + 트리트먼트(2막 전반) + 씬 리스트(2a 씬들) + 이미지 시스템 + 캐릭터
**출력**: `step_07_first_draft_act2a.md`

**→ 사용자 점검.** 2막 전반 집필 결과를 사용자에게 제시하고, 승인 또는 수정 요청을 받는다.

### 호출 3: 2막 후반(2b) 집필

**입력**: 1막+2a 요약 + 트리트먼트(2막 후반) + 씬 리스트(2b 씬들) + 이미지 시스템 + 캐릭터
**출력**: `step_07_first_draft_act2b.md`

**→ 사용자 점검.** 2막 후반 집필 결과를 사용자에게 제시하고, 승인 또는 수정 요청을 받는다.

### 호출 4: 3막 집필

**입력**: 1막+2막 요약 + 트리트먼트(3막) + 씬 리스트(3막 씬들) + 이미지 시스템 + 캐릭터
**출력**: `step_07_first_draft_act3.md`

**→ 사용자 점검.** 3막 집필 결과를 사용자에게 제시하고, 승인 또는 수정 요청을 받는다.

### 호출 5: 이음새 검토 + 합본

4개 막 모두 사용자 승인을 받은 뒤에 진행한다.

**입력**: 각 막의 마지막 2씬 + 다음 막의 첫 2씬 (이음새 3곳)
**작업**:
- 톤과 분위기의 연결이 자연스러운지 확인한다.
- 이음새 부분만 수정한다. 전체를 다시 쓰지 않는다.
- 4개 파일을 하나로 합본한다.

**출력**: `step_07_first_draft_full.md` (전체 합본)

## 집필 원칙

### 지문 쓰기

1. **Tony Tost — 보이지 않는 숏 리스트.** 모든 문장이 하나의 숏이고, 마침표마다 컷이다. 한 문장 안에 두 개의 숏을 넣지 않는다.
   ```
   예시 (축척 변화로 카메라 이동을 암시한다):
   "이 도시 변두리의 경비 삼엄한 식민지풍 저택."   → 와이드 숏
   "지잔(40대)이 정문을 지키고 있다."                → 미디엄 숏
   "지잔은 아이폰을 보며 히죽 웃고 있다."           → 클로즈업
   "그것은 로드러너 루니 툰스 만화다."               → 인서트 숏
   ```

2. **Trottier — 스크린에 나타날 수 없는 것은 쓰지 않는다.**
   - ❌ "그는 외롭다" → ✅ "편의점 도시락을 혼자 먹는다"
   - ❌ "오래 앉아 있었다" → ✅ "열 번째 담배를 비벼 끈다"
   - ❌ "과거를 회상하며" → ✅ 회상 씬을 삽입하거나 현재 행동으로 보여준다

3. **지문은 최대 3~5줄로 쓴다.** 길어지면 단락을 나눈다. 긴 묘사 블록은 읽는 흐름을 끊는다.

4. **현재 시제, 능동태로 쓴다.** "문이 열린다"라고 쓰고, "문이 열렸다"라고 쓰지 않는다.

5. **McKee — 구체적인 단어를 고른다.** "큰 못"이라고 쓰지 말고 "대못"이라고 쓴다. 형용사+일반명사 대신, 그 자체로 정확한 단어를 사용한다.

6. **카메라 지시는 쓰지 않는다.** PAN, ZOOM, CUT TO 같은 기술 용어는 감독의 영역이다. 대신 묘사 방식으로 카메라를 암시한다:
   - 작은 디테일을 묘사하면 클로즈업을 암시한다.
   - 광대한 환경을 묘사하면 와이드 숏을 암시한다.
   - 새 문단을 시작하면 새 숏이나 컷을 암시한다.
   - 짧은 문장 파편은 인서트를 암시한다. ("사진 한 장. 깨진 유리.")

7. **묘사를 읽는 시간이 곧 장면을 보는 시간이다.** 긴박한 장면은 짧고 빠른 문장으로 쓰고, 느린 장면은 천천히 읽히도록 쓴다.

### 대사 쓰기

8. **인물이 실제로 이렇게 말할지 생각한다.** 현실적인 구어체를 사용한다. 문어체 대사는 어색하다.

9. **서브텍스트를 유지한다.** 인물이 말하는 것과 인물이 원하는 것은 다르다. 이 간극이 대사에 힘을 준다.

10. **대사로 정보를 전달하지 않는다.** "너도 알다시피, 우리 엄마가 치과를 운영하시잖아"처럼 등장인물이 이미 아는 정보를 대사로 설명하는 것은 게으른 방법이다.

## 분량 가이드 (90분 기준)

한국 시나리오는 A4 1페이지가 약 1분 30초에 해당한다. 90분 영화는 약 60~70페이지가 된다.

**주요 지표는 씬 수다.** 90분 영화는 70~85씬으로 구성한다. 글자 수와 페이지 수는 보조 지표로 참고한다.

### 글자 수 기준

시나리오는 씬 헤딩, 대사 들여쓰기, 문단 사이 여백 등으로 산문(트리트먼트)보다 페이지당 글자 수가 적다. A4 한 페이지에 약 **500자**가 들어간다.

| 목표 | 순수 본문 글자 수 |
|------|-----------------|
| 60페이지 (최소) | 30,000자 이상 |
| 65페이지 (기준) | 약 32,500자 |
| 70페이지 (최대) | 35,000자 이하 |

- "순수 본문"이란 마크다운 기호를 제외한 시나리오 텍스트(씬 헤딩 + 지문 + 대사)를 뜻한다.
- 각 막 집필을 마친 뒤 글자 수를 확인하고, 이전 막 요약 아래에 기입한다.
- **막당 7,000자 미만이면 씬이 너무 얇게 쓰인 것이다.** 대지문과 소지문이 충분한지 점검한다.

### 막별 분량 가이드

| 막 | 씬 수 | 글자 수 기준 | 페이지 (참고) | 상세도 |
|----|--------|-------------|--------------|--------|
| 1막 | 18~21씬 | 7,000~9,000자 | 약 13~18페이지 | 셋업은 간결하게, 전환점은 상세하게 쓴다 |
| 2막 전반 | 18~21씬 | 7,000~9,000자 | 약 13~18페이지 | Fun & Games의 밀도를 높이고, 미드포인트는 상세하게 쓴다 |
| 2막 후반 | 18~21씬 | 7,000~9,000자 | 약 13~18페이지 | 위기를 고조시키고, 간결한 씬을 많이 쓴다 |
| 3막 | 16~22씬 | 6,000~8,500자 | 약 11~16페이지 | 클라이맥스를 가장 상세하게 쓰고, 에필로그는 간결하게 쓴다 |

## 품질 기준

- [ ] 한국 시나리오 형식을 따르고 있는가? (씬 헤딩, 대지문/소지문 구분, 대사 왼쪽 들여쓰기)
- [ ] 대지문과 소지문이 제 역할을 하고 있는가? 대지문은 공간을, 소지문은 인물의 행동을 묘사하는가?
- [ ] 대사가 왼쪽 들여쓰기로 되어 있는가? 중앙 정렬(할리우드 형식)을 쓰지 않았는가?
- [ ] 카메라에 담을 수 없는 서술이 없는가? ("그는 슬프다" 등)
- [ ] 지문이 3~5줄 이하로 끊어져 있는가?
- [ ] 현재 시제, 능동태로 작성되었는가?
- [ ] 여백의 미학이 지켜지고 있는가? 빽빽한 텍스트 블록은 없는가?
- [ ] 이미지 시스템의 모티프가 자연스럽게 등장하는가?
- [ ] 막간 이음새가 자연스러운가?
- [ ] 총 씬 수가 70~85씬 범위 안에 있는가?
- [ ] 막별 글자 수(공백 제외)가 최소 기준 이상인가? (1막/2a/2b: 7,000자, 3막: 6,000자)

## 주의사항

- 트리트먼트(STEP 5)는 길잡이이지 대본이 아니다. 트리트먼트를 그대로 옮기지 않는다.
- 첫 집필이므로 완벽할 필요는 없다. STEP 8에서 시각적 개고가 이어진다.
- 할리우드 형식과 한국 형식을 섞지 않는다. 한국 시나리오 형식으로 통일한다.
