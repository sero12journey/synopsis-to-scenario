# Critic → Writer — STEP 8: 시각적 개고 (Visual Revision)

## 개요

이 Step은 두 단계로 진행된다:
1. **Critic이 진단**: Linda Seger 시각적 개고 프로세스로 1고를 분석
2. **Writer가 반영**: Critic의 진단을 바탕으로 수정

## 입력 컨텍스트

다음 파일을 읽어라:
- `projects/{name}/output/step_07_first_draft_full.md` — 시나리오 1고
- `projects/{name}/output/step_03_image_system.md` — 이미지 시스템 설계
- `projects/{name}/output/step_00_critique.md` — 보완점 목록 (최종 대조)

상세 진단 기준이 필요하면 참조:
- `영화시나리오연구_영화적묘사_markdown.md` — Linda Seger 시각적 개고, 카메라 불가능 서술 패턴, 서브텍스트 번역 사례

## Phase 1: Critic 진단

### Linda Seger 시각적 개고 6단계

1고를 처음부터 끝까지 읽으며 다음 6가지를 진단하라:

### 1. 대사 과다 장면 식별 (Linda Seger)

- 대사가 행동을 대체하고 있는 장면을 찾아라
- "대사로 설명하는 것 → 시각적 행동으로 대체 가능한 것" 목록 작성
- 각 장면의 씬 번호, 현재 대사, 시각적 대안 제안
- David Mamet 테스트: "등장인물이 말을 못 한다고 가정하고 무성영화를 써라" — 대사 없이도 장면이 성립하는가?

### 2. 반복되는 시각 요소 식별

- 의도적 반복(이미지 시스템)과 비의도적 반복 구분
- STEP 3의 이미지 시스템과 대조하여 일관성 확인
- 빠진 모티프 등장 지점 또는 과다 등장 지점 식별

### 3. 설명적 지문 탐지

- "감정/심리를 직접 서술한" 지문을 찾아라
- 각각에 대해 영상적 행동 변환 제안:
  - ❌ "그는 외롭다" → ✅ "편의점 도시락을 혼자 먹는다"
  - ❌ "그녀는 분노했다" → ✅ "유리잔을 움켜쥔 손가락이 하얗게 변한다"
  - ❌ "그는 충격을 받았다" → ✅ "스카치를 따르지만 손이 떨려 대부분이 잔 바깥으로 흐른다"
  - ❌ "그는 각오가 서 있다" → ✅ "시가에 불을 붙인다. 손이 완벽히 안정되어 있다" (《대부》)
- **추가 탐지 대상**: 3~4줄 초과 액션 라인 블록, 추상적 형용사+일반명사 조합 ("큰 못" → "대못")

### 4. 카메라 불가능 서술 제거

- 화면에 담을 수 없는 서술을 찾아라 (Trottier: "스크린에 나타날 수 없는 것은 쓸 수 없다"):
  - ❌ "그는 오래 앉아 있었다" → ✅ "열 번째 담배를 비벼 끈다"
  - ❌ "과거를 회상하며" → ✅ (회상 씬을 삽입하거나, 현재 행동으로 번역)
  - ❌ "그녀는 오래된 슬픔을 느꼈다" → ✅ (카메라에 포착 가능한 행동으로 번역)
  - ❌ "그들은 서로 사랑했다" → ✅ (행동, 시선, 물리적 근접/거리로 번역)
- **McKee 기준**: "언어적 표현의 90%는 영화적 등가물이 없다" — 남은 10%만이 시나리오에 쓸 수 있는 것

### 5. 복선-회수 최종 점검

- STEP 5의 복선-회수 매핑과 대조하여:
  - 심은(Plant) 복선이 1고에 실제로 있는가
  - 회수(Payoff)가 1고에 실제로 있는가
  - 누락된 것 목록

### 6. 이미지 시스템 상징적 상승 확인

- STEP 3의 상징적 상승 계획과 대조하여:
  - 초반: 모티프가 구체적 사물로 등장하는가
  - 중반: 반복으로 무게가 생겼는가
  - 후반: 보편적 공명으로 변환되었는가
  - Opening Image ↔ Final Image 대비가 실현되었는가

## Phase 2: Writer 반영

Critic의 진단 결과를 바탕으로 1고를 수정하라.

### 수정 규칙

1. 진단 항목을 하나씩 처리
2. 대사 → 시각적 행동 변환 시, 씬의 흐름이 자연스러운지 확인
3. 모티프 추가/수정 시, 과하지 않게 (잠재의식적 수준 유지 — McKee: 관객이 의식하면 설교)
4. 복선 누락 시, 가장 자연스러운 위치에 삽입
5. 설명적 지문 → 영상적 행동 번역 시 Tony Tost 원칙 적용 (문장 = 숏, 축척 변화로 카메라 이동)
6. 전체 수정 후, 수정 전후 대비 요약 작성

## 최종 점검

모든 수정이 끝난 후:
- [ ] 카메라 불가능 서술이 0개인가
- [ ] 모든 Plant에 Payoff가 있는가
- [ ] 이미지 시스템 모티프가 3개 막에 분포하는가
- [ ] Opening Image ↔ Final Image가 대비/대응하는가
- [ ] 총 분량이 55~70페이지인가
- [ ] STEP 0 보완점이 최대한 반영되었는가
