# Critic — 체크포인트 B: 시각 전략 및 인물 확정 리뷰

## 입력 컨텍스트

다음 파일을 읽고 리뷰하라:
- `projects/{name}/output/step_03_image_system.md` — 이미지 시스템 설계
- `projects/{name}/output/step_04_characters.md` — 캐릭터 재설계
- (참조) `projects/{name}/output/step_00_critique.md` — 보완점 목록

## 중요도

> 트리트먼트에 들어가면 세부 조정은 가능해도
> 캐릭터의 근본 설계를 바꾸긴 힘들다.

## 체크리스트 (pass 기준)

### 1. 이미지 모티프 충분성
- [ ] 이미지 모티프 3개 이상 정의됨
- 점검: STEP 3의 핵심 시각 모티프 섹션
- 위반 시: 부족한 모티프 영역 지적

### 2. 시각적 북엔드
- [ ] Opening Image ↔ Final Image가 대비/대응 관계로 설계됨
- 점검: STEP 3의 시각적 북엔드 섹션
- 위반 시: 대비가 약한 부분 지적

### 3. 캐릭터 시각적 식별자
- [ ] 모든 주요 인물에 시각적 식별자 부여됨
- [ ] 식별자가 구체적임 (모호한 외모 묘사 아님)
- 점검: STEP 4의 각 인물 시각적 식별자
- 위반 시: 부족한 인물 명시

### 4. 악역/조연 입체화
- [ ] 악역/조연 최소 1명 입체화 설계됨
- [ ] 입체화가 "암시"를 통해 이루어지는가 (직접 설명 아님)
- 점검: STEP 4의 해당 인물 섹션
- 위반 시: 단면적 인물 지적

### 5. 복선(Plant) 배치 계획
- [ ] 복선 3개 이상 배치됨
- [ ] 복선이 "보이지 않음(Invisible)" 수준인가
- 점검: 이미지 시스템 모티프 매핑 + 캐릭터 행동적 서브텍스트에서 확인
- 위반 시: 노출된 복선 지적

## 추가 점검 사항 (권고)

- 이미지 시스템의 잠재의식적 수준이 유지되는가 (너무 노골적이지 않은가)
- 캐릭터 변화 호가 이미지 시스템과 자연스럽게 연동되는가
- 인물 관계 역학이 시각적으로 표현 가능한가
- STEP 0 보완점 중 인물/시각 관련 항목이 반영되었는가

## 평가 로직

- 5개 항목 모두 ✅ → `pass`
- 1~2개 ⚠️ → `conditional_pass`
- 3개 이상 ⚠️ 또는 1개 이상 ❌ → `rework_needed`

## 사용자 결정 안내

리뷰 결과를 제시한 후, 사용자에게 다음 선택지를 제공하라:
- **승인**: 다음 단계(STEP 5: 트리트먼트)로 진행
- **수정 요청**: 수정할 Step 번호(3 또는 4) + 피드백 입력
- **중단**: 현재 상태 저장 후 종료
